# בקרת ליקויי בניה

MVP מקומי לביקורי שטח, משימות וליקויים מתמשכים בין ביקורים. הממשק בעברית RTL, הנתונים נשמרים ב-IndexedDB, והדו״ח מיועד להדפסה או שמירה כ-PDF דרך הדפדפן.

## הרצה מקומית

```bash
npm install
npm run dev
```

## בדיקות ובניה

```bash
npm run typecheck
npm run test -- --run
npm run build
npm run test:e2e
```

## עיקרי ה-MVP

- פרויקט וביקורים מרובים ללא שרת וללא התחברות.
- משימות ביקור עם תבנית בדיקות נפוצה.
- רשם ליקויים ברמת הפרויקט, כולל סטטוס, חומרה, אחראי, יעד ותמונות.
- ליקויים פתוחים ובטיפול ממשיכים להופיע בביקורים חדשים עד סימון כטופל/אומת/סגור.
- ליקויים שטופלו נשארים בהיסטוריית הפרויקט ואינם נמחקים.
- דו״ח ביקור שמפריד בין חדשים, נמשכים מביקורים קודמים, טופלו בביקור, ועדיין פתוחים.
- PWA בסיסי עם manifest ו-service worker.
